# TravelIQ

游析通是一个旨在通过对旅游数据分析为用户提供个性化旅游推荐服务的前后端分离项目，实现与优化了用户点评、统计、推荐、词云等功能，提升了用户体验和系统性能。

# 项目启动

```
pip install -r requirements.txt
```

# 后端

```
cd .\backend\
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```

# 前端

```
cd .\frontend\
npm start
```
